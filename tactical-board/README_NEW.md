# 戦術ボードツール

シンプルで静かなWebベースの戦術ボードです。10×10のグリッド上にコマを自由に配置し、矢印で動きを示せます。

## 機能

### 🎯 コマ（4種類）
- **A（赤）**、**B（青）**、**C（黄）**、**D（紫）** の4種類
- クリックでグリッド上に**自由配置**（グリッドにスナップしない）
- ドラッグ＆ドロップで位置変更可能
- **右クリックで削除**

### 📍 描画機能
1. **自由描画モード**
   - マウスで自由に線を描画
   - 戦術図の詳細な説明に使用

2. **直線矢印モード**
   - 始点をクリック → 終点をクリック
   - 方向や移動を明確に示す

3. **消しゴムモード**
   - 線に触れると、その線全体が消える（部分消しではない）
   - 自由描画と矢印の両方に対応

### ✨ その他の特徴
- **保存機能なし** - 常に初期状態から開始
- **LocalStorage未使用** - データは保存されません
- **GitHub Pages対応** - 外部サーバー不要

## 使用方法

### セットアップ

#### 方法1: GitHub Pages で公開
1. このリポジトリを GitHub に push
2. Settings → Pages で main ブランチを選択
3. `https://[username].github.io/[repo-name]/` にアクセス

#### 方法2: ローカルで実行
```bash
cd /tactical-board
python -m http.server 8080
# ブラウザで http://localhost:8080 にアクセス
```

### 基本的な使い方

#### コマを配置する
1. 左パネルの「コマを配置」セクションで、A/B/C/Dのいずれかを選択
2. グリッド上の配置したい位置をクリック
3. コマが自由な位置に配置されます（グリッドスナップなし）

#### コマを移動する
1. 配置済みのコマをマウスで選択してドラッグ
2. 新しい位置に移動

#### コマを削除する
1. 削除したいコマを右クリック
2. コマが削除されます

#### 自由描画で線を引く
1. 「自由描画」モードに切り替え
2. グリッド上をマウスでドラッグして線を描画
3. マウスを離すと描画が確定

#### 直線矢印で動きを示す
1. 「直線矢印」モードに切り替え
2. 矢印の始点をクリック
3. 矢印の終点をクリック
4. 矢印が描画されます

#### 描画を削除する
1. 「消しゴム」モードに切り替え
2. 削除したい線や矢印をクリック
3. **その線全体が削除**されます

## 技術仕様

### グリッド
- 10×10のセル構成
- 1セルは60×60ピクセル
- 静かで控えめな線色（#e5ddd5）

### コマ
- ピクセル座標で自由配置（グリッドスナップなし）
- 半径30ピクセルの距離判定でドラッグ可能
- 右クリック削除対応

### 描画設定
- 描画線の幅：3px
- 描画線色：#7f8c8d（グレー）
- 矢印色：#3498db（ブルー）
- 消しゴムサイズ：15px

### 色設定
- A: #ff6b6b（レッド）
- B: #4ecdc4（シアン）
- C: #f9ca24（イエロー）
- D: #6c5ce7（パープル）
