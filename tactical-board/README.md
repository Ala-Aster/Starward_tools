# 戦術ボードツール

シンプルで静かなWebベースの戦術ボードです。10×10のグリッド上にコマを配置し、矢印で動きを示せます。

## 機能

### 🎯 コマ（4種類）
- **A（赤）**、**B（青）**、**C（黄）**、**D（紫）** の4種類
- クリックでグリッド上に配置
- ドラッグ＆ドロップで位置変更可能

### 📍 描画機能
1. **自由描画モード**
   - マウスで自由に線を描画
   - 戦術図の詳細な説明に使用

2. **直線矢印モード**
   - 始点をクリック → 終点をクリック
   - 方向や移動を明確に示す

3. **消しゴムモード**
   - 間違った描画を削除
   - 一部のみ消すことが可能

### 💾 データ管理
- **LocalStorage対応** - ブラウザに自動保存
- **保存ボタン** - 手動保存
- **リセット** - データをクリアして初期状態に戻す
- **戻す（Undo）** - Ctrl+Z で最後の操作を取り消し

## 使用方法

### セットアップ
1. このフォルダをWebサーバーで公開するか、ローカルでindex.htmlを開く
2. ブラウザで http://localhost:PORT/index.html にアクセス

### 基本的な使い方

#### コマを配置する
1. 左パネルの「コマを配置」セクションで、A/B/C/Dのいずれかを選択
2. グリッド上の配置したい位置をクリック
3. コマが配置されます

#### コマを移動する
1. 配置済みのコマをマウスで選択
2. ドラッグして新しい位置に移動

#### 自由描画で線を引く
1. 「自由描画」モードに切り替え
2. グリッド上をマウスでドラッグして線を描画
3. マウスを離すと描画が確定

#### 直線矢印で動きを示す
1. 「直線矢印」モードに切り替え
2. 矢印の始点をクリック
3. 矢印の終点をクリック
4. 矢印が描画されます

#### 描画を削除する
1. 「消しゴム」モードに切り替え
2. 削除したい線や矢印をクリック
3. その位置から消しゴムサイズ分が削除されます

### データ保存
- **自動保存** - 操作する度にブラウザのLocalStorageに保存
- **保存ボタン** - 明示的に保存
- **リセット** - すべてのデータをクリア

## 技術仕様

### グリッド
- 10×10のセル構成
- 1セルは60×60ピクセル
- 静かで控えめな線色（#e5ddd5）

### 描画設定
- 描画線の幅：3px
- 描画線色：#7f8c8d（グレー）
- 矢印色：#3498db（ブルー）
- 消しゴムサイズ：15px

### 色設定
- A: #ff6b6b（レッド）
- B: #4ecdc4（シアン）
- C: #f9ca24（イエロー）
- D: #6c5ce7（パープル）

## ブラウザ互換性
- Chrome/Edge（推奨）
- Firefox
- Safari

## 注意事項
- データはブラウザのLocalStorageに保存されます
- キャッシュをクリアするとデータが削除されます
- 複数タブでの同時編集はサポートしていません

## デザインの特徴
- シンプルで落ち着いた配色
- 余計なアニメーションを排除
- 視覚ノイズを最小化
- 操作性に焦点を当てたUIレイアウト
